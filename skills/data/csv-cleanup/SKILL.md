# SKILL: csv-cleanup

## Trigger
CSV/엑셀 데이터 정제 요청 (결측·중복·형식 불일치 등).

## Procedure
1. 스키마와 샘플 행을 먼저 확인한다 (열 의미·타입).
2. 결측/중복/이상치를 진단해 사용자에게 보고한다.
3. 정제 규칙을 합의한 뒤 적용한다 (원본은 보존, 사본 작업).
4. 처리 전후 행 수·요약 통계를 비교한다.
5. 결과 파일을 지정 폴더에 저장.

## Pitfalls
- 원본 덮어쓰기 → 항상 사본으로 작업.
- 무단 행 삭제 → 규칙을 먼저 합의하고 건수 보고.

# SKILL: chart-gen

## Trigger
데이터 시각화/차트 생성 요청.

## Procedure
1. 전달하려는 메시지와 데이터 형태를 확인한다.
2. 메시지에 맞는 차트 유형을 고른다 (추세=선, 비교=막대, 비중=파이 등).
3. 축 라벨·단위·범례·제목을 반드시 넣는다.
4. 이미지/파일로 저장하고 경로를 알려준다.

## Pitfalls
- 축 라벨/단위 누락 → 생성 후 점검 필수.
- 메시지에 안 맞는 차트 유형 → 유형 선택을 먼저.

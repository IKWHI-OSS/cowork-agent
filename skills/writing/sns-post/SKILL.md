# SKILL: sns-post

## Trigger
SNS 게시물 작성 요청 (트위터/X, 링크드인, 인스타 캡션 등).

## Procedure
1. 플랫폼·타깃·목적(정보전달/홍보/참여)을 확인한다.
2. 핵심 메시지 1개로 압축한다.
3. 플랫폼 길이·톤에 맞춰 초안 + 변형 2안을 만든다.
4. 해시태그/CTA를 덧붙인다.
5. USER.md의 톤 선호를 반영한다.

## Pitfalls
- 한 게시물에 메시지 여러 개 → 1게시물 1메시지.
- 플랫폼 길이 제한 초과 → 작성 후 글자수 점검.
